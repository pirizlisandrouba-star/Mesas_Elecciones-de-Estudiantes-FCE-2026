# Mesas - Elecciones de Estudiantes FCE 2026

Aplicación web simple para consultar información de mesas de votación de estudiantes.

## 🚀 Acceso a la app

Podés usar la aplicación desde cualquier dispositivo en el siguiente link:

👉 https://pirizlisandrouba-star.github.io/Mesas_Elecciones-de-Estudiantes-FCE-2026/

## 📱 Uso

* Ingresar los datos solicitados
* Consultar la mesa asignada
* Visualizar la información de forma rápida y clara
